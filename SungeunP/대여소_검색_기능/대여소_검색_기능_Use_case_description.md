# Use Case Description

### 대여소 검색 기능

| **Actor Action**                                | **System Response**                             |
| ----------------------------------------------- | ----------------------------------------------- |
| 1. None                                         | 2. 대여소 검색을 위한 화면을 출력한다.          |
| 3. 회원은 대여소 검색을 위한 키워드를 입력한다. | 4. 조건에 맞게 검색된 대여소 리스트를 출력한다. |
